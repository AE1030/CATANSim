# CATANSim
[![Quality Gate Status](https://sonarcloud.io/api/project_badges/measure?project=elsayyay_CATANSim&metric=alert_status)](https://sonarcloud.io/summary/new_code?id=elsayyay_CATANSim)

A Java-based Settlers of Catan simulator developed for SFRWENG 2AA4 at McMaster University. The simulator supports a mixed game of one human player and multiple AI opponents, complete with board visualization via a JSON state export.

run javac *.java,

then java Demonstrator
